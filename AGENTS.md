# One Page

单页作品集。每个子目录都是一个独立的单页项目，可直接在浏览器中打开。

---

## 工具

| 项目 | 简介 |
|------|------|
| [LOGO 语言模拟器 (Logo Simulator)](logo-sim/) | 一个基于浏览器的 LOGO 编程语言模拟器，支持编写和执行 LOGO 指令控制海龟画图。 |
| [图片调整大小工具 (Pic Tool)](pic-tool/) | 一个纯前端的图片尺寸调整工具，支持拖拽上传图片、自定义宽高比例、预览和下载调整后的图片。 |

## 互动体验

| 项目 | 简介 |
|------|------|
| [指尖星河 (Finger Galaxy)](finger-galaxy/) | 手势/鼠标控制的星空互动页面。通过手指或鼠标在 Canvas 上绘制星尘轨迹，呈现沉浸式的银河交互体验。 |
| [鱼戏莲叶间 (Gesture Fish)](gesture-fish/) | 基于 MediaPipe 手势识别的锦鲤池塘互动页面。支持鼠标和摄像头手势双模式：滑动水面产生涟漪，悬停/长按投喂鱼食，60 条锦鲤在莲叶间自由游弋。 |
| [云端都市 (Kimi City)](kimi-city/) | 基于 Canvas 的交互式城市体验，支持拖拽和滚轮缩放视角，呈现赛博朋克风格的都市夜景。 |
| [三维水管 (Pipes)](pipes/) | 经典 Windows「三维水管」屏保的 WebGL2 重现。纯黑底 + 白/黄/红三色水管在 3D 网格上随机生长，拐弯处带球形关节；支持拖拽旋转视角，滚轮缩放，静置后缓慢自动环绕；长满后自动清场重来。 |

## 项目结构

```
one-page/
├── index.html          # 主页（由 sync.sh 生成）
├── AGENTS.md           # 本文件（项目说明）
├── sync.sh             # 同步脚本，扫描子目录生成主页
```

## 主页生成规则

主页 `index.html` 由 `sync.sh` 脚本自动生成，规则如下：

### 扫描规则

1. 扫描根目录下所有子目录（忽略以 `.` 开头的目录）
2. 每个子目录必须包含 `AGENTS.md` 文件才会被收录
3. 从 `AGENTS.md` 中解析以下信息：
   - **标题**：第一行 `# ` 开头的内容
   - **类型**：`> 类型：xxx` 格式的内容
   - **简介**：第一个非标题、非引用、非空行的文本

### 生成内容

`sync.sh` 会生成两个文件：

1. **index.html**：可视化主页
   - 按类型分组展示所有项目
   - 每个项目显示为卡片，包含标题和简介
   - 点击卡片跳转到对应子目录的 `index.html`
   - 支持深色/浅色模式自动切换

2. **根目录 AGENTS.md**：Markdown 格式的项目列表
   - 按类型分组的表格形式展示
   - 包含项目名称和简介

### 使用方式

```bash
# 单次运行，生成主页
./sync.sh

# 监听模式，文件变化时自动重新生成
./sync.sh --watch
```

## 添加新项目

1. 在根目录下创建新的子目录
2. 在子目录中创建 `index.html` 作为项目主页
3. 在子目录中创建 `AGENTS.md`，格式如下：

```markdown
# 项目名称

> 类型：作品展示

这里写项目的简介描述。

## 技术栈

- 技术1
- 技术2

## 使用方式

直接在浏览器中打开 `index.html` 即可。
```

4. 运行 `./sync.sh` 更新主页

## 项目类型

目前收录的项目类型包括：

- **工具**：实用工具类项目
- **互动体验**：交互式体验项目
- **游戏**：游戏类项目
- **作品展示**：文学、影视、游戏等作品的介绍页面

## 注意事项

- 子目录中的 `AGENTS.md` 是必须的，用于生成主页列表
- 项目类型会自动按首字母排序分组
- 简介文本会截取第一个有效段落（最多显示约100字符）
