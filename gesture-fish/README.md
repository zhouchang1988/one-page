# 鱼戏莲叶间 (Gesture Fish)

基于 MediaPipe 手势识别的锦鲤池塘互动页面。支持鼠标和摄像头手势双模式：滑动水面产生涟漪，悬停/长按投喂鱼食，60 条锦鲤在莲叶间自由游弋。

## 技术栈

- Canvas 2D 渲染
- MediaPipe Hands 手势识别（可选）
- 支持鼠标和触摸手势输入

## 功能

- 手势/鼠标双模式切换
- 涟漪效果与鱼食投喂系统
- 多种锦鲤花色和 AI 行为
- 莲叶、莲花、水下光斑等场景元素
- 多款书法字体可切换

## 使用方式

直接在浏览器中打开 `index.html` 即可。点击"开启摄像头"按钮启用手势控制。
