# One Page

单页作品集。每个子目录都是一个独立的单页项目，可直接在浏览器中打开。

---

## 工具

| 项目 | 简介 |
|------|------|
| [LOGO 语言模拟器 (Logo Simulator)](logo-sim/) | 一个基于浏览器的 LOGO 编程语言模拟器，支持编写和执行 LOGO 指令控制海龟画图。 |
| [图片调整大小工具 (Pic Tool)](pic-tool/) | 一个纯前端的图片尺寸调整工具，支持拖拽上传图片、自定义宽高比例、预览和下载调整后的图片。 |

## 互动体验

| 项目 | 简介 |
|------|------|
| [指尖星河 (Finger Galaxy)](finger-galaxy/) | 手势/鼠标控制的星空互动页面。通过手指或鼠标在 Canvas 上绘制星尘轨迹，呈现沉浸式的银河交互体验。 |
| [鱼戏莲叶间 (Gesture Fish)](gesture-fish/) | 基于 MediaPipe 手势识别的锦鲤池塘互动页面。支持鼠标和摄像头手势双模式：滑动水面产生涟漪，悬停/长按投喂鱼食，60 条锦鲤在莲叶间自由游弋。 |

## 游戏

| 项目 | 简介 |
|------|------|
| [技能五子棋 (Skills Five)](skill-five/) | 一个带有特殊技能的五子棋游戏，支持双人对战和人机对战，基于纯前端技术实现，无需后端服务器。 |

## 作品展示

| 项目 | 简介 |
|------|------|
| [给阿嬷的情书 (Dear You)](dear-you/) | 一封献给阿嬷的电子情书，温暖怀旧的单页页面，配有背景音乐，以细腻的视觉设计表达对祖母的深情。 |
| [幻城 (Huan Cheng)](huan-cheng/) | 郭敬明玄幻经典小说《幻城》的单页介绍页面，以冰雪主题呈现小说中的奇幻世界观、人物和故事线。 |
| [明朝那些事儿 (Ming Chao)](ming-chao/) | 当年明月历史著作《明朝那些事儿》的单页介绍页面，以明朝风格的视觉设计展示明朝历史脉络与精彩章节。 |
| [搜神记与蛮荒记 (Sou Shen Ji)](sou-shen-ji/) | 树下野狐神话史诗《搜神记》与《蛮荒记》的单页介绍页面，以深邃的神话风格呈现远古洪荒世界观、人物与故事。 |
| [仙逆 (Xian Ni)](xian-ni/) | 耳根修仙小说《仙逆》的单页介绍与解析页面，以仙侠风格呈现"顺为凡，逆则仙"的修仙世界观、境界体系与故事脉络。 |
| [藏海传 (Zang Hai Zhuan)](zang-hai-zhuan/) | 权谋复仇传奇《藏海传》的单页介绍页面，以暗色金饰风格呈现小说中的权谋世界与复仇故事线。 |
